# Create QR Code for PDF in C# with Syncfusion

The Syncfusion .NET PDF Library allows developers to generate and embed QR codes into PDF documents seamlessly using C#. This guide showcases a curated set of samples demonstrating how to create basic and customized QR codes, integrate branding elements, position QR codes in headers and footers, reuse them across existing documents, and export them as images for broader use cases.

Sample name | Description
------|------
[Embed a Basic QR Code in a PDF using C#](https://github.com/SyncfusionExamples/add-qrcode-in-pdf-csharp/tree/master/embed-qrcode-in-new-pdf-file) | Demonstrates how to generate and add a simple QR code to a PDF document. Ideal for encoding URLs, text, IDs, or contact details.
[Customize QR Code Appearance](https://github.com/SyncfusionExamples/add-qrcode-in-pdf-csharp/tree/master/customize-qr-code-appearance) | Shows how to customize QR code properties such as size, X-dimension, error correction level, and quiet zone to meet scanning and design requirements.
[Add QR Codes to PDF Headers and Footers](https://github.com/SyncfusionExamples/add-qrcode-in-pdf-csharp/tree/master/embed-qrcodes-in-pdf-headers-and-footers) | Explains how to place QR codes consistently in headers or footers across all PDF pages, useful for tracking, verification, or navigation.
[Overlay a Logo on a QR Code (Branding)](https://github.com/SyncfusionExamples/add-qrcode-in-pdf-csharp/tree/master/qrcode-with-logo) | Demonstrates how to embed a logo or brand image within a QR code while maintaining scan reliability.
[Add a QR Code to an Existing PDF Document](https://github.com/SyncfusionExamples/add-qrcode-in-pdf-csharp/tree/master/add-qrcode-in-existing-pdf) | Shows how to insert QR codes into an already generated PDF without recreating the document from scratch.
[Export a QR Code as an Image](https://github.com/SyncfusionExamples/add-qrcode-in-pdf-csharp/tree/master/export-qrcode-as-image) | Explains how to generate a QR code as an image file for reuse across PDFs, web pages, emails, or printed materials.
[Real‑World Use Cases (Event Tickets & Invitations, Invoices & Receipts)](https://github.com/SyncfusionExamples/add-qrcode-in-pdf-csharp/tree/master/realword-usecases-create-qrcode-in-pdf) | Shows how QR codes enable quick ticket validation for events and provide easy access to payment links or invoice details in receipts and bills.

## How to run the examples

* Download this project to a location in your disk.
* Open the solution file using Visual Studio.
* Rebuild the solution to install the required NuGet package.
* Run the application.
